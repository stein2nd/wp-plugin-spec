> 各プラグインリポジトリ直下の `docs/` フォルダ等にコピーして利用することを想定しています。

* **目的**: `## 8. REST API 仕様` をプラグイン固有のエンドポイント設計に具体化する。
* **内容例**:
  * `/plugin-slug/v1/...` ごとの

    * 認可ポリシー・`permission_callback` 条件

* **AI 的メリット**:
  * 「このエンドポイントの PHP 実装を作って」「このレスポンスを使うフロントのフックを書いて」といったタスクを、**1エンドポイント単位で安全に委譲できる**。
