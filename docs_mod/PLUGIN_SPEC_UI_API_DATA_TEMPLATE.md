## 4. AI 伴走開発での使い方メモ

- UI 実装タスク:
  - 本ファイルの「1. 画面仕様 (UI) とフロー」を AI に渡し、対象画面 ID を明示してコンポーネント生成を依頼する。
- API 実装タスク:
  - 「2. REST API 仕様・外部連携」の対象エンドポイント節を指定し、`includes/RestController.php` 等の実装を依頼する。
- データ移行・リファクタリング:
  - 「3. データ辞書」を AI に渡し、「option_name/meta_key を変更するが、既存データの移行コードも提案してほしい」のように依頼する。
